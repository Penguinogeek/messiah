Messiah is open for developers to use our data. We offer a RESTful API to develop apps for developers. See documentation below:

Path Args Method Use
/history City/Country GET Lists the history of past year disasters.
/show_random_facts - GET Show some random fact based on past year data static as in our homepage.
/random_facts - GET Returns {Deaths, Year, Type} randomly in raw format.
/predict_eq_mag Lattitude, Longitude, Depth, Date(optional) GET Returns magnitude of possible earthquake in current or given date.
